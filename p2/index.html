Assignment #3: Font Size, Family, and Color

Created by Austin Nahouray

In order to create an aesthetically pleasing webpage, it is helpful to know how to change font size, family, and color. This webpage will teach and explain how to create and change each of these elements.

Font Size

To change the font size in CSS, you must state, "font-size: --%;"

Example code: 
  body3{   
   font-size: 200%;
    }

Font Family

To change the font family in CSS, you must state, "font-family: "---, ---, ---;"

Example code:

body5{ font-family: "---", ---, ---; }

Font Color

To change the font color in CSS, you must state, "color:---;"

Example code:

body7{ color:blue; }
